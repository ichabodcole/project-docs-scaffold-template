# Grapevine V1.6.7 — presence honesty, grounding, consume-path paper cuts

**Date:** 2026-05-28

Shipped the V1.6.7 patch (toolbox `2.8.0 → 2.9.0`) from the V1.6 multi-channel
roundtable: honest presence counts (`connections`/`named`/`anonymous`, so a
`watch` tab no longer reads as a ghost), `who --all` + `GET /presence`, a
`doctor` count-vs-names cross-check, a `tail` `kind:"grounding"` on-connect
line, a `: grapevine-keepalive` stderr tick, `truncation_hint` re-ordered before
`.text` with the threshold raised 800 → 2000, and a `send` target echo. A TDD
spike disproved the presumed flagship "reaping is broken" bug — reaping already
works (SIGKILL'd consumer reaped ≤ ~6s), so `idleTimeout` was left untouched and
the real fix was count semantics. Full suite 52/52 + a 5/5 live 2-agent smoke;
independent review cleared it. `reply`/`@mention`/`announce`/identity deferred
to V1.7.
